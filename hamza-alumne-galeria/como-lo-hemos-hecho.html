Desarrollo del proyecto

Este proyecto ha sido desarrollado utilizando HTML y CSS para crear una galería de cartas de futbolistas. Cada carta contiene información detallada sobre un jugador específico, incluyendo su biografía, características y logros.

Estructura del proyecto

El proyecto está organizado en varios archivos HTML, cada uno representando una carta individual.Tambien dos archivos CSS para dar estilo a la pagina princpal y otro CSS que edita cada uno de los detalles de la carta.Despues en referencias y en como lo hemos hecho he utilizado la etiqueta style para no tener que crear otro CSS.

Herramientas utilizadas

Se han utilizado Visual Studio Code para escribir codigo , wikipedia para sacar la la biografia de los jugadores i Chat GPT para validar informacion para sacar fuentes de informacion y donde esta cada informacion en la wikipedia.Tambien he utilizado live server para ver la pagina localmente y he utilizado indent rainbow para ver los tab con mas claridad.
